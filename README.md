## Getting started

**Technologies**

-JavaScript
-React
-SCSS

**Requirements**

- Node.js 16.0.0+
- npm 7.10.0+ **\*\*** PH ****\*\*\*\*****

**First steps**

1. Clone this repository
2. Run `npm install`
3. Run `npm start`

## Scripts

### `npm run start`

Start the develepoment enviroment.

### `npm run build`

Build the project for production.
